# 1.3 Relationship Tags and Segments

**Area:** 1. Core Relationship Management

Users can categorize people into groups or segments.

## Capabilities

- Add custom tags to profiles.
- Filter by tags.
- Create groups.
- Use tags in reminders and suggestions.

## Example Tags

- close family
- college friends
- work
- VIP
- gift ideas
- long-distance
- needs more attention
- sensitive relationship
- clients

## Possible Data Objects

- `Tag`
- `RelationshipTagging`
- `RelationshipGroup`

## Implementation Notes

Tags will help with search, reminders, and future batch actions.
