# 2.2 Desires, Wants, and Future Ideas

**Area:** 2. Relationship Memory

Users can track things someone has mentioned wanting, needing, or hoping to do.

## Capabilities

- Add wants/desires manually.
- Extract wants from notes.
- Convert wants into gift ideas or plans.
- Mark desires as fulfilled.
- Track fulfillment history.

## Examples

- wants to travel to Japan
- wants new headphones
- wants to try pottery
- wants to go camping
- wants to visit a specific restaurant
- wants help organizing the garage
- wants a spa day

## Possible Data Objects

- `Desire`
- `DesireCategory`
- `DesireStatus`
- `DesireFulfillment`

## Implementation Notes

This feature should feed into gift recommendations, date ideas, birthday planning, and spontaneous gesture suggestions.
