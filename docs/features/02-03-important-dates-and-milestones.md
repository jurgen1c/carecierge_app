# 2.3 Important Dates and Milestones

**Area:** 2. Relationship Memory

Users can track meaningful dates connected to each relationship.

## Capabilities

- Add important dates.
- Support recurring and one-time dates.
- Add reminder schedules.
- Add date importance levels.
- Link dates to planning workflows.
- Add cultural/religious/local holiday support later.

## Date Types

- Birthday
- Anniversary
- Graduation
- Wedding
- Memorial date
- Medical appointment
- Interview
- Exam
- Performance
- First day of school
- Work milestone
- Religious holiday
- Cultural holiday
- Custom

## Possible Data Objects

- `ImportantDate`
- `DateType`
- `ReminderRule`
- `ReminderInstance`

## Implementation Notes

Important dates should not only trigger reminders. They should also trigger planning opportunities.

Example: birthday in 21 days should suggest gift planning.
