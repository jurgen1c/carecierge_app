# 2.4 Relationship Timeline

**Area:** 2. Relationship Memory

Each relationship has a chronological timeline of meaningful events, notes, interactions, gifts, commitments, and plans.

## Capabilities

- Display all relationship activity chronologically.
- Filter timeline by type.
- Add manual timeline entries.
- Automatically add system events.
- Link timeline entries to gifts, events, reminders, and notes.

## Timeline Entry Types

- Note
- Conversation recap
- Gift
- Event
- Promise
- Reminder
- Conflict
- Apology
- Important date
- Plan
- Booking
- AI extraction
- User correction

## Possible Data Objects

- `TimelineEntry`
- `TimelineEntryType`
- Polymorphic reference to source object

## Implementation Notes

The timeline should become the relationship history backbone.
