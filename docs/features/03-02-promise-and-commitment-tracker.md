# 3.2 Promise and Commitment Tracker

**Area:** 3. Reminders and Commitments

Users can track things they promised or intend to do for someone.

## Capabilities

- Add commitments manually.
- Extract commitments from notes.
- Assign due dates.
- Mark as complete.
- Link to reminders.
- Link to relationship timeline.
- Track overdue commitments.

## Examples

- Send article to John.
- Help dad set up phone.
- Take son biking.
- Check in after Maria's appointment.
- Follow up with boss about report.

## Possible Data Objects

- `Commitment`
- `CommitmentStatus`
- `CommitmentReminder`

## Implementation Notes

This is one of the highest-value features because forgotten commitments create real relationship damage.
