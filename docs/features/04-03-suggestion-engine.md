# 4.3 Suggestion Engine

**Area:** 4. AI Assistance and Intelligence

The app recommends timely actions based on relationship data, dates, preferences, goals, and commitments.

## Capabilities

- Suggest gifts.
- Suggest messages.
- Suggest plans.
- Suggest check-ins.
- Suggest date ideas.
- Suggest event ideas.
- Suggest conflict repair actions.
- Suggest professional follow-ups.
- Explain why suggestions were made.
- Allow feedback.

## Suggestion Types

- Reminder-based
- Occasion-based
- Preference-based
- Goal-based
- Commitment-based
- Spontaneous
- Repair-focused
- Professional follow-up

## Possible Data Objects

- `Suggestion`
- `SuggestionType`
- `SuggestionReason`
- `SuggestionFeedback`
- `SuggestionStatus`

## Implementation Notes

Suggestions should always include reasoning.

Example: suggested because her birthday is in 12 days and she recently mentioned wanting to try pottery.
