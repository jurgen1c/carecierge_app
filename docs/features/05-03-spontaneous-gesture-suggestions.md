# 5.3 Spontaneous Gesture Suggestions

**Area:** 5. Daily Experience

The app suggests small, thoughtful actions outside of major occasions.

## Capabilities

- Suggest low-effort actions.
- Suggest medium-effort actions.
- Suggest high-effort actions.
- Match suggestions to relationship type.
- Match suggestions to recent emotional context.
- Track whether action was taken.

## Examples

- Send a voice note.
- Order favorite dessert.
- Send an article.
- Schedule lunch.
- Bring coffee.
- Send encouragement.
- Plan a walk.
- Mail a handwritten note.

## Possible Data Objects

- `GestureSuggestion`
- `GestureCategory`
- `GestureFeedback`

## Implementation Notes

This feature should feel warm and useful, not forced.
