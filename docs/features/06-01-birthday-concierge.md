# 6.1 Birthday Concierge

**Area:** 6. Planning Workflows

A guided workflow for planning birthdays.

## Capabilities

- Detect upcoming birthdays.
- Ask for budget and preferences.
- Generate celebration concepts.
- Recommend gifts.
- Suggest vendors.
- Build checklist.
- Draft invitations.
- Track RSVPs.
- Build day-of timeline.
- Create reminders.
- Store post-event recap.

## Workflow Steps

1. Upcoming birthday detected.
2. User starts planning.
3. App reviews profile and preferences.
4. App asks constraints.
5. App generates options.
6. User selects plan.
7. App creates tasks and reminders.
8. User executes manually or through future automation.
9. App records outcome.

## Possible Data Objects

- `BirthdayPlan`
- `PlanConcept`
- `PlanTask`
- `PlanBudget`
- `PlanGuest`
- `PlanVendorOption`

## Implementation Notes

This should be a flagship feature. MVP can create plans and checklists before real booking integrations exist.
