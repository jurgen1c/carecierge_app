# 6.2 Anniversary Concierge

**Area:** 6. Planning Workflows

A guided workflow for anniversaries and romantic milestones.

## Capabilities

- Detect upcoming anniversaries.
- Suggest date ideas.
- Suggest gifts.
- Suggest flowers.
- Suggest restaurants.
- Suggest personal message.
- Create timeline.
- Support childcare reminder.
- Support reservation flow later.

## Possible Data Objects

- `AnniversaryPlan`
- Reuse generic `Plan`, `PlanTask`, `GiftIdea`, `VendorOption`

## Implementation Notes

This can reuse the generic event planning system with anniversary-specific prompts.
