# 6.4 Personal Touch Checklist

**Area:** 6. Planning Workflows

Every plan should distinguish between logistics the app can help with and personal actions only the user should provide.

## Capabilities

- Generate personal touch suggestions.
- Require user input for personal details.
- Prompt for handwritten note.
- Prompt for inside joke.
- Prompt for meaningful memory.
- Prompt for toast or private message.
- Track completion.

## Example Sections

App can help:

- reservation
- cake
- flowers
- reminders
- invitation draft

User should handle:

- personal note
- meaningful memory
- emotional presence
- final message approval

## Possible Data Objects

- `PersonalTouchItem`
- `PersonalTouchPrompt`

## Implementation Notes

This feature reinforces the product philosophy.
