# 7.2 Vendor Shortlisting and Comparison

**Area:** 7. Vendor, Booking, and Marketplace Layer

The app compares vendor options for a specific plan.

## Comparison Criteria

- Price
- Availability
- Location
- Reviews
- Style match
- Dietary support
- Cancellation policy
- Package details
- Delivery options
- Past user feedback
- Persona fit

## Possible Data Objects

- `VendorOption`
- `VendorComparison`
- `VendorScore`
- `VendorMatchReason`

## Implementation Notes

The app should explain why each vendor is recommended.
