# 9.1 Response Suggestion Assistant

**Area:** 9. Communication and Social Context

The app suggests responses to messages, situations, or social posts.

## Capabilities

- User pastes message or situation.
- App suggests response options.
- App adapts tone to relationship.
- User edits and approves.
- Store final message optionally.

## Response Types

- Supportive
- Funny
- Romantic
- Professional
- Apologetic
- Encouraging
- Boundary-setting
- Celebratory
- Concise

## Possible Data Objects

- `ResponseSuggestion`
- `ResponseContext`
- `ResponseTone`

## Implementation Notes

Avoid automatic sending in early versions.
