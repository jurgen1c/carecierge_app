# 9.2 Social Media Context Helper

**Area:** 9. Communication and Social Context

Users can manually provide a social media post or screenshot and ask for interpretation or response suggestions.

## Capabilities

- Analyze manually provided content.
- Suggest tone-aware response.
- Suggest whether to respond at all.
- Store relevant memory with approval.

## Possible Data Objects

- `SocialContextItem`
- `SocialResponseSuggestion`

## Implementation Notes

Avoid covert monitoring or platform scraping. Keep this user-initiated at first.
