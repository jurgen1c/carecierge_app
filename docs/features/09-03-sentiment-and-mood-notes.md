# 9.3 Sentiment and Mood Notes

**Area:** 9. Communication and Social Context

Users can record perceived emotional states or concerns.

## Capabilities

- Add mood note.
- Link mood note to relationship timeline.
- Create follow-up reminder.
- Suggest supportive action.
- Track recurring patterns.

## Examples

- seemed stressed
- seemed distant
- seemed excited
- seemed sad
- seemed overwhelmed
- seemed proud

## Possible Data Objects

- `MoodNote`
- `MoodCategory`
- `MoodFollowUp`

## Implementation Notes

This should not pretend to diagnose anyone. It is for user observation and follow-up.
