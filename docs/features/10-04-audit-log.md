# 10.4 Audit Log

**Area:** 10. Privacy, Safety, and Control

The app records important user and system actions.

## Logged Actions

- Profile updated
- AI memory extracted
- Reminder created
- Message drafted
- Vendor contacted
- Booking requested
- Purchase approved
- Automation rule used
- Sensitive record accessed
- Data exported
- Data deleted

## Possible Data Objects

- `AuditLog`
- `AuditEventType`
- `Actor`
- `AuditMetadata`

## Implementation Notes

This helps with transparency and debugging.
