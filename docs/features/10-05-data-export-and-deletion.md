# 10.5 Data Export and Deletion

**Area:** 10. Privacy, Safety, and Control

Users can export or delete their data.

## Capabilities

- Export relationship profile.
- Export all data.
- Export timeline.
- Export calendar dates.
- Delete profile.
- Delete account.
- Delete sensitive notes.
- Delete AI-generated data.

## Export Formats

- JSON
- CSV
- PDF summary
- Calendar file

## Possible Data Objects

- `ExportJob`
- `DeletionRequest`

## Implementation Notes

Data portability should be considered early.
