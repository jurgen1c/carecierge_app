# 11.1 Calendar Integration

**Area:** 11. Integrations

Sync important dates, reminders, events, and plans with user calendars.

## Capabilities

- Import birthdays from calendar.
- Create calendar events.
- Detect upcoming meetings.
- Trigger relationship briefings.
- Add event planning tasks.
- Remind before important dates.

## Possible Integrations

- Google Calendar
- Apple Calendar
- Outlook

## Implementation Notes

Calendar integration is high-value but can be Phase 2 or Phase 3.
