# 11.2 Contacts Integration

**Area:** 11. Integrations

Import people from contacts and enrich relationship profiles.

## Capabilities

- Import name, phone, email.
- Match existing profiles.
- Add birthdays if available.
- Avoid duplicates.
- Let user choose which contacts to import.

## Possible Integrations

- Google Contacts
- Apple Contacts
- Phone contacts

## Implementation Notes

Contacts import can reduce onboarding friction but must be permission-sensitive.
