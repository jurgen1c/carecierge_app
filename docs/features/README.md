# Carecierge Features

## Product Definition

**Carecierge** is a private relationship concierge that helps users remember, plan, and follow through for the people who matter. It combines relationship memory, reminders, AI assistance, planning workflows, and permission-based logistics automation.

The product should be designed around this principle:

> **Automate logistics. Augment care. Keep the human in control.**
